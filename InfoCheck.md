# 정보확인

## 네트워크 정보 확인

- ip 주소 확인

```bash
ipconfig
```



## 디렉토리 용량 확인

- 파일 용량까지 바이트 단위로

```bash
ls -al
```

- 인간 옵션

```bash
ls -alh
```



> File System 및 SSD에서 최소 저장 사이즈는 4KB
>
> 따라서 6 Byte를 저장하더라도, 빈 공간으로 꽉 채워 4KB 단위로 저장한다.



- 디스크 용량 확인 

```bash
du ./bts.txt
```

```bash
du -h ./bts.txt
```
