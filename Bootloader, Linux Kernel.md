# 부트로더

## Bootloader

- 부팅시 동작되는 프로그랩
- Disk에 저장되어 있는 운영체제를 실행시키는 역할
- 우분투는 GRUP2, 윈도우는 Bootmgr



## 기능

- 다중 OS 부팅
- 장치 제어 및 테스트
- HW 관련 코드가 들어있고, 제어할 수 있다.
